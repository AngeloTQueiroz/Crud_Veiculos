# Crud_Veiculos
Crud de venda de veículos com interface Web usando Spring Boot



para acessar a aplicação precisa rodar ela na sua IDE, e acessar no seu navegador:  http://localhost:8080/index.html

para visualizar API de clientes cadastrados : http://localhost:8080/clientes


os demais endereços são possiveis serem acessados a partir do index.

o projeto ainda esta em desenvolvimento, atualmente ele faz as funcões de cadastrar, editar, listar e excluir um cliente. Faltando apenas replicar isso para os veiculos, e criar a interação de compra de veiculo.

A ideia era criar um array como atributo para a classe Cliente onde fosse possivel armazenar os ID dos carros comprados pelo mesmo. Dessa forma era masi facil integrar as funcionalidades CRUD.
